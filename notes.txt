$ git init - create a new git repo^C

$ git add -A - Add all modified files in working area to staging area

$ git commit - perform a commit (Store previous version from staging area to current area)

$ git log - Display all the commits of the repository

$ git status - display current status of the repo

$ git restore - recover file from stagging area

$ git revert - recover file from commit area

$ git branch - Display branches

$ git branch <branch name> - Create a new branch by given name

$ git branch -m - rename a branch

$ git branch -d - delete a branch if merged

$ git branch -D - delete a branch even if not merged
